说明：这是学习和复盘框架，不构成投资建议。

## 一句话理解

鱼盆模型是趋势和轮动仪表盘，不是交易指令。当前市场状态只维护在 `outputs/maobidao-yupen-action-board.md`，本文只沉淀可复用的方法变化。

它更适合回答：

- 当前哪个指数/板块更强？
- 强势方向有没有持续？
- 市场是从防守转向进攻，还是从进攻转向分化？
- 哪些板块正在从后排进入前排？

它不适合单独回答：

- 明天买什么？
- 某个点位一定涨还是跌？
- 某个板块是否具备长期投资价值？

## 鱼盆表格看什么

核心字段可以拆成五层：

| 层级 | 看什么 | 含义 | 常见误区 |
|---|---|---|---|
| 方向 | 是否站上20日线 | 判断是否处于趋势上方 | 站上均线不等于立刻能追 |
| 强度 | 偏离率 | 趋势强弱和拥挤程度 | 偏离率越高，短线越可能波动 |
| 时间 | 状态转变时间 | 趋势持续多久 | 刚转强和转强很久不能同样处理 |
| 排名 | 横向排名 | 相对其他指数/板块强不强 | 单日排名不能代表长期主线 |
| 变化 | 排名变化 | 资金是否持续迁移 | 只看当天，不看连续变化 |

## 推荐读表流程

```text
1. 先看宏观和流动性
2. 再看指数鱼盆
3. 再看板块鱼盆
4. 最后看连续排名变化
```

### 1. 先看宏观和流动性

鱼盆是价格结果，宏观和流动性是背景。

例如本轮：

```text
美伊缓和
→ 油价压力下降
→ 通胀压力下降
→ 加息预期下降
→ 风险偏好修复
→ A股成长指数转强
```

如果没有宏观背景，只看鱼盆，会知道“变强了”，但不知道为什么变强。

### 2. 再看指数鱼盆

指数鱼盆先回答市场风格：

- 海外强还是A股强？
- 成长强还是价值强？
- 科创/创业强，还是沪深300/上证50强？
- 港股有没有跟随？

本轮核心变化：

- 6月初：海外强，A股弱。
- 6月中：海外先转强，A股成长随后修复。
- 6月下旬：科创50、中证500、创业板、中证A500进入前排。
- 7月初：科创50仍强，但创业板、中证A500、沪深300分化。

### 3. 再看板块鱼盆

板块鱼盆回答资金具体流向：

- 半导体是否仍在前排？
- 创新药、证券、机器人是否连续上升？
- 老主线是正常调整，还是被新主线替代？
- 有没有板块从强势区滑落？

本轮核心变化：

- 6月初：煤炭、红利阶段性靠前，科技回撤。
- 6月中：半导体、证券、有色逐步修复。
- 6月下旬：半导体继续强，创新药和证券开始抬头。
- 7月初：创新药、证券、机器人、商业航天进入观察区。

### 4. 最后看连续排名变化

连续变化比单日排名更重要。

更有价值的信号：

- 科创50从后排升到第1。
- 中证500持续进入前排。
- 恒生科技长期靠后。
- 创业板从第3跌到第14/16。
- 创新药从观察区升到板块前排。

这些比“某天涨了多少”更有信息量。

## 用鱼盆做决策时的边界

鱼盆可以作为中线过滤器：

```text
只在趋势强、排名靠前、连续改善的方向里找机会。
```

但它不能替代：

- 行业逻辑
- 估值判断
- 业绩兑现
- 政策风险
- 仓位管理
- 止盈止损规则

更稳的决策链条：

```text
宏观环境允许
→ 鱼盆确认方向
→ 行业逻辑能解释
→ 排名连续改善
→ 仓位分批进入
→ 趋势破坏时退出或降仓
```

## 操作倾向字典

鱼盆表格最终要落到动作，否则就会停留在“知道但不会用”。建议只用 6 个动作，别搞复杂。

| 动作 | 触发条件 | 含义 |
|---|---|---|
| 主攻 | 连续前排，排名稳定，宏观环境配合 | 当前阶段最值得跟踪的方向。 |
| 持有 | 已在强势方向里，排名仍靠前但偏离率不低 | 不轻易下车，但也不盲目加仓。 |
| 观察 | 后排开始连续改善，但还没进入前排 | 加入清单，等二次确认。 |
| 试探 | 进入前排并保持，但仍缺乏充分验证 | 小仓、分批、轻仓验证，不重仓押注。 |
| 降仓 | 前排方向排名快速下滑，或关键均线/趋势破坏 | 降低进攻暴露，避免从趋势单变成信仰单。 |
| 回避 | 长期后排，反弹也无法进入前排 | 不因便宜或跌多就参与。 |
