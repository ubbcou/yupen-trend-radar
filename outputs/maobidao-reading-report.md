## 总览

作者的核心价值不是单日预测，而是连续复盘。方法大致是：

- 短线看情绪、消息、外围夜盘。
- 中线看鱼盆模型、均线趋势、板块强弱。
- 长线看流动性、时间价值、仓位管理。
- 单篇文章常有生活闲聊，但市场主线始终围绕“复盘、纪律、周期匹配”展开。

### 鱼盆使用框架

鱼盆不回答“明天买什么”，它回答“当前资金和趋势偏向哪里”。更稳的用法是把它放在交易链条中间：

```text
宏观/流动性判断
→ 鱼盆确认主线和相对强弱
→ 连续观察排名变化
→ 区分短线反弹和中线趋势
→ 再决定仓位、周期和退出规则
```

五个读表层次：

1. 方向：是否站上20日线，红绿只是第一层。
2. 强度：偏离率越高，趋势越强，但也越容易短线拥挤。
3. 时间：刚转强和转强很久，含义不同。
4. 排名：同一时点横向比较，谁更强。
5. 变化：连续多日排名上升/下降，比单日排名更重要。

本轮最有价值的鱼盆信号不是“科创50某天第1”，而是：

- 科创50从6月中旬后排持续升到第1，说明A股科技主线被确认。
- 中证500跟随进入前排，说明行情不只集中在单一科创指数。
- 恒生科技长期后排，说明港股科技没有跟随A股科技同步修复。
- 7月初创业板、中证A500、沪深300下滑，但科创50仍第1，说明强势内部开始分化。
- 创新药、证券、机器人、商业航天在板块表中抬头，是板块轮动预警。

## 横向主题汇总

### 1. 鱼盆模型

鱼盆模型在报告里承担的是“中线纪律系统”，不是神谕。

它被用于：

- 判断指数和板块是否处于趋势中。
- 识别强弱排序。
- 避免单靠感觉追涨杀跌。
- 提醒趋势坏了要退出。

作者反复强调：模型胜率可以不高，但只要能少数几次吃到大趋势、避开大跌，就有价值。

### 2. AI主线

AI在文章中有两层含义：

- 股票主线：半导体、存储、光通信、算力、数据中心。
- 个人生产力：AI工具、WorkBuddy、代码生成、回测、知识库、工作原型。

这是理解作者的重要点：他不是只把AI当题材炒，而是把AI当成时代工具使用。所以他对AI长期共识偏乐观，但对短期硬件估值和二阶预期会转谨慎。

### 3. 流动性

美伊、油价、通胀、加息预期是6月中旬行情修复的关键链条：

```text
美伊缓和
→ 油价压力下降
→ 通胀预期下降
→ 加息预期下降
→ 流动性预期改善
→ 风险资产修复
```

这条链条解释了为什么作者在6月中旬从谨慎转向更积极。

### 4. 板块轮动

板块轮动的关键变化：

- 5月底：科技回调，消费/地产试探反弹。
- 6月中：科技重新占优。
- 6月底：半导体仍强，创新药开始出现。
- 7月初：证券、创新药、机器人进入前排，AI硬件出现二阶预期裂缝。

后续最应该盯的不是单日涨跌，而是鱼盆模型前排能否连续变化。

### 5. 交易纪律

作者最反复强调的纪律：

- 短线、中线、长线不能混。
- 博反弹失败不能改成长期价值投资。
- 单一个股信仰危险。
- 高位不敢买会错过主线，但高位追买也要有退出规则。
- 仓位管理比单次判断更重要。
