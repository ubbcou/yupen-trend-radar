# 鱼盆模型学习指南

- 生成时间：2026-07-06
- 最后迭代时间：2026-07-20
- 迭代来源：2026-07-20 前已整理文章
- 用途：沉淀鱼盆模型读法、误区修正和动作规则，辅助维护操作看板

说明：这是学习和复盘框架，不构成投资建议。

## 一句话理解

鱼盆模型是趋势和轮动仪表盘，不是交易指令。当前市场状态只维护在 `docs/action-board.md`，本文只沉淀可复用的方法变化。

它更适合回答：

- 当前哪个指数/板块更强？
- 强势方向有没有持续？
- 市场是从防守转向进攻，还是从进攻转向分化？
- 哪些板块正在从后排进入前排？

它不适合单独回答：

- 明天买什么？
- 某个点位一定涨还是跌？
- 某个板块是否具备长期投资价值？

## 鱼盆表格看什么

核心字段可以拆成五层：

| 层级 | 看什么 | 含义 | 常见误区 |
|---|---|---|---|
| 方向 | 是否站上20日线 | 判断是否处于趋势上方 | 站上均线不等于立刻能追 |
| 强度 | 偏离率 | 趋势强弱和拥挤程度 | 偏离率越高，短线越可能波动 |
| 时间 | 状态转变时间 | 趋势持续多久 | 刚转强和转强很久不能同样处理 |
| 排名 | 横向排名 | 相对其他指数/板块强不强 | 单日排名不能代表长期主线 |
| 变化 | 排名变化 | 资金是否持续迁移 | 只看当天，不看连续变化 |

## 推荐读表流程

```text
1. 先看宏观和流动性
2. 再看技术结构和指数共振
3. 再看消息催化和文章主观判断是否有独立逻辑
4. 再看指数鱼盆
5. 再看板块鱼盆
6. 最后看连续排名变化和偏离率风险
```

### 1. 先看宏观和流动性

鱼盆是价格结果，宏观和流动性是背景。

例如本轮：

```text
美伊缓和
→ 油价压力下降
→ 通胀压力下降
→ 加息预期下降
→ 风险偏好修复
→ A股成长指数转强
```

如果没有宏观背景，只看鱼盆，会知道“变强了”，但不知道为什么变强。

### 2. 再看技术结构和指数共振

技术结构用于判断大盘有没有顺风环境。

需要关注：

- 关键点位是否守住，例如整数关口、前低、20日线、BOLL中轨/下轨。
- 多个指数图形是否相近，如果上证、沪深、创业、科创同步修复，共振更强。
- 外部利好是否被快速消化，如果利好高开低走，说明合力仍偏弱。

技术结构不替代鱼盆，但能防止只看板块强势而忽略大盘压力。

#### 短线反弹与中线趋势隔离

同一个交易日可以同时出现两种不冲突的判断：

```text
短线：急跌后技术超卖，可能反弹1至2天
中线：鱼盆仍在20日线下，趋势买入条件不成立
```

本项目只消费第二种信号。短线反弹可以作为市场环境记录，但不能替代鱼盆前三、站上20日线、强于大盘，也不能把“回避”直接升级为“试探”。只有反弹后重新满足完整条件，才重新进入趋势方向池。

#### BOLL 与鱼盆的关系

默认参数下，BOLL 中轨通常就是20日均线；鱼盆临界值也近似围绕20日趋势阈值构造。因此：

- “鱼盆转红”和“站上BOLL中轨”通常反映的是同一类趋势信息，不能当成两次独立确认。
- BOLL上轨和下轨可以提示短线位置是否极端，但突破上下轨不自动构成高抛或抄底指令。
- “均值上下两个标准差约覆盖95.45%”依赖正态分布等假设，不等于价格触及下轨后有95.45%的反弹胜率。
- 本项目仍以鱼盆前三、站上20日线、强于大盘为方向判断核心；BOLL只补充短线位置和风险，不改变方向分组。

### 3. 再看消息催化和文章主观判断是否有独立逻辑

文章判断不能替代鱼盆，但可以作为独立证据。

更有价值的情况是：

- 作者基于技术结构、外围情绪或产业逻辑给出方向判断。
- 鱼盆表格在同一天或相邻数据日出现同向变化。
- 两条证据不是互相抄答案，而是从不同路径得到相似结论。

例如 2026-07-09 文章中，猫哥主观判断港股走出主跌浪，而鱼盆表格显示恒生科技、国企指数、恒生指数同步转红并进入前五。这种同向信号可以把港股从等待升为试探，但仍要等连续前排和回踩不破20日线，不能直接升为主攻。

2026-07-10 的信号更强：BOLL 下轨反弹提供技术位置，长鑫存储上市、高盛看多中国AI、国产替代 Plan B 提供消息导火线，鱼盆上科创50第1且放量、半导体第1且放量。这种组合可以升级为主攻方向。

但要额外检查偏离率。科创50偏离率 11.54%、半导体偏离率 9.23%，说明趋势很强，也说明短线拥挤度高。方向上可以确认主线，动作上不能自动等同于追高。

2026-07-10 数据进一步说明标签必须逐日复算：科创50和半导体仍排前二、仍站上20日线，但当日均明显弱于中证A500，因此从主攻降为趋势保持；CS创新药升到第1并满足全部确认条件，接替成为主攻方向。前一天的结论不能因为排名表面稳定而自动延续。

2026-07-13 数据把这条规则推得更彻底：科创50从指数第1降到第5并跌破20日线，半导体从板块第2降到第7并跌破20日线。即使国产替代、AI扩产等叙事仍在，趋势买入也要先退出新增框架。大跌后可以有反弹，但只有重新满足三条件并出现连续性，才有资格回到方向池。

### 4. 再看指数鱼盆

指数鱼盆先回答市场风格：

- 海外强还是A股强？
- 成长强还是价值强？
- 科创/创业强，还是沪深300/上证50强？
- 港股有没有跟随？

本轮核心变化：

- 6月初：海外强，A股弱。
- 6月中：海外先转强，A股成长随后修复。
- 6月下旬：科创50、中证500、创业板、中证A500进入前排。
- 7月初：科创50仍强，但创业板、中证A500、沪深300分化；7 月 8 日港股三杰转红；7 月 9 日科创50继续第1并放量，国产替代主线重新压过宽基修复；7 月 13 日A股主流指数全部跌破临界值，趋势买入进入空窗期。

### 5. 再看板块鱼盆

板块鱼盆回答资金具体流向：

- 半导体是否仍在前排？
- 创新药、证券、机器人是否连续上升？
- 老主线是正常调整，还是被新主线替代？
- 有没有板块从强势区滑落？

本轮核心变化：

- 6月初：煤炭、红利阶段性靠前，科技回撤。
- 6月中：半导体、证券、有色逐步修复。
- 6月下旬：半导体继续强，创新药和证券开始抬头。
- 7月初：创新药、证券、机器人、商业航天进入观察区；7 月 9 日半导体重新升为第1并放量，和科创50形成指数与板块共振；7 月 13 日半导体跌到第7并跌破20日线，创新药和消费保留前排但需要量能和文章逻辑继续确认。

### 6. 最后看连续排名变化和偏离率风险

连续变化比单日排名更重要。

更有价值的信号：

- 科创50从后排升到第1。
- 中证500持续进入前排。
- 恒生科技长期靠后。
- 港股三杰单日转红只能算试探，必须看下一次是否连续前排。
- 科创50第1叠加半导体第1，比单一指数第1更有主线意义。
- 创业板从第3跌到第14/16。
- 创新药从观察区升到板块前排。
- 全绿后是否重新站回20日线，比单日低开反弹更重要。
- 偏离率过高时，先确认方向，再等待更好的动作位置。
- 曾经的主线一旦跌破20日线，必须重新走完整确认流程，不能靠旧叙事保留买入资格。

这些比“某天涨了多少”更有信息量。

## 用鱼盆做方向判断时的边界

本项目只专精趋势买入。鱼盆作为方向过滤器，只回答这一件事：

```text
某个指数或板块是否进入趋势买入方向池。
```

但它不能替代：

- 行业逻辑
- 估值判断
- 业绩兑现
- 政策风险

更稳的方向判断链条：

```text
宏观环境允许
→ 技术结构不逆风
→ 鱼盆前排
→ 站上20日线
→ 所属板块强于大盘
→ 进入方向池
→ 记录下一次验证点
```

## 趋势方向分组字典

只保留 5 个分组。

| 分组 | 触发条件 | 含义 |
|---|---|---|
| 主攻 | 满足产品流程中的完整主攻确认条件 | 当前最值得跟踪的趋势方向；触发偏热条件时附加“不追高”。 |
| 试探 | 核心趋势成立，但至少一项确认条件待满足 | 进入观察，等待缺失条件确认。 |
| 趋势保持 | 趋势仍在，但当前不满足新增条件 | 趋势未破，不新增。 |
| 等待 | 只有部分条件改善 | 继续观察下一次鱼盆表和文章。 |
| 回避 | 同时满足产品流程中的后排和水下条件 | 不纳入趋势买入方向。 |

前排、偏离率、市场基准、连续性、量能和偏热的固定数值口径统一引用 `docs/product-flow.md`，本文不另设阈值。

当前数据的方向判断统一维护在 `docs/action-board.md`。
