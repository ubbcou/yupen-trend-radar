## 使用方式

这份报告不按自然月整理，而按“市场驱动因子”整理：

```text
宏观环境
→ 流动性预期
→ 市场风险偏好
→ 产业主线
→ 板块轮动
→ 指数/鱼盆趋势
→ 仓位和交易纪律
```

每篇文章保留五类信息：

- 核心观点
- 市场背景
- 板块/趋势数据
- 作者判断
- 后续验证点

阶段总结用于压缩上下文，方便后续新增文章时判断：它是在延续旧阶段，还是开启新阶段。

## 总览

作者的核心价值不是单日预测，而是连续复盘。方法大致是：

- 短线看情绪、消息、外围夜盘。
- 中线看鱼盆模型、均线趋势、板块强弱。
- 长线看流动性、时间价值、仓位管理。
- 单篇文章常有生活闲聊，但市场主线始终围绕“复盘、纪律、周期匹配”展开。

当前最重要的连续线索：

- AI 与半导体是 2026 上半年最强主线。
- 6 月中旬美伊缓和改善油价、通胀和加息预期，是风险偏好修复的关键。
- 7 月初开始从“AI硬件强趋势”转向“AI硬件二阶预期有裂缝”，同时加入技术结构和指数共振验证。
- 7 月 13 日鱼盆数据确认A股主流指数和半导体、科创50同步破位，趋势买入进入空窗期。
- 7 月 23 日上证50和创新药跌出前三；煤炭连续第1但接近过热线且缺文章确认，红利新进第2但缩量，当前仍无主攻。科技长期产业逻辑转强，但IPO前的短期资金风险与鱼盆弱势一致。

### 鱼盆使用框架

鱼盆不回答“明天买什么”，它回答“当前资金和趋势偏向哪里”。更稳的用法是把它放在交易链条中间：

```text
宏观/流动性判断
→ 鱼盆确认主线和相对强弱
→ 连续观察排名变化
→ 区分短线反弹和中线趋势
→ 再决定仓位、周期和退出规则
```

五个读表层次：

1. 方向：是否站上20日线，红绿只是第一层。
2. 强度：偏离率越高，趋势越强，但也越容易短线拥挤。
3. 时间：刚转强和转强很久，含义不同。
4. 排名：同一时点横向比较，谁更强。
5. 变化：连续多日排名上升/下降，比单日排名更重要。

本轮最有价值的鱼盆信号不是“科创50某天第1”，而是：

- 科创50从6月中旬后排持续升到第1，说明A股科技主线被确认。
- 中证500跟随进入前排，说明行情不只集中在单一科创指数。
- 恒生科技长期后排后在 7 月 8 日升到第2，并在 7 月 9 日维持第2，说明港股右侧修复仍在，但暂时不是本篇主线。
- 7月初创业板、中证A500、沪深300下滑，但科创50仍第1，说明强势内部开始分化。
- 7月9日半导体重新升到板块第1并放量，和科创50第1形成指数与板块共振，是国产替代主线回归信号。
- 7月10日CS创新药升到板块第1并满足完整主攻条件；科创50和半导体虽仍排前二，但因当日弱于中证A500而失去新增条件，说明方向标签必须随每个数据日重算。
- 7月14日科创50从No转回Yes却仍只排第6、弱于中证A500且缩量；这进一步说明“站上20日线”只是三条件之一，不能单独升级为趋势买入。
- 同日CS创新药完整趋势条件成立，但偏离率达到9.12%且文章中性，因此只能试探并标记“不追高”；趋势方向与买点动作必须分开。
- 7月21日科创50和半导体出现两位数附近的大涨，文章也支持国产算力，但二者仍是后排水下；短线BOLL/MACD时点和长期产业逻辑都不能覆盖鱼盆方向资格。
- 7月22日上证50首次进入前三并站上20日线，但前次第4、量比0.96且文章中性；这是“可试探”而不是“已反转”，下一期连续性比单日名次更重要。

## 横向主题汇总

### 1. 鱼盆模型

鱼盆模型在报告里承担的是“中线纪律系统”，不是神谕。

它被用于：

- 判断指数和板块是否处于趋势中。
- 识别强弱排序。
- 避免单靠感觉追涨杀跌。
- 提醒趋势坏了要退出。

作者反复强调：模型胜率可以不高，但只要能少数几次吃到大趋势、避开大跌，就有价值。

7月13日新增的关键方法认识是：鱼盆临界值和BOLL中轨都近似20日趋势阈值，不能当作两份独立证据重复加权。BOLL上下轨只用于辅助识别短期极端位置，方向池仍以鱼盆前三、20日线和相对大盘强弱为核心。

7月14日进一步验证了纪律价值：7月10日的国产替代强共振没有被自动外推成持续主线，7月13日科创50和半导体同时跌出趋势买入条件。7月14日科创50虽重新转Yes，但排名、相对强弱和量能仍未恢复；对本项目来说，单一转红不能替代完整三条件。

7月15日数据把这种独立校验推进了一步：作者已经基于鱼盆信号定投红利，但中证红利当日仅排第4，所以项目仍将其列为等待；相反，中证煤炭完成连续前排并满足主攻条件。文章态度是确认项，不替代前三、20日线和相对强弱。

7月17日数据增加了时间周期边界：作者可以同时认为“短线超跌可抄”和“中线鱼盆为空头”，两者并不矛盾。由于本项目只做趋势买入，1至2天的暴跌反弹不进入方向池；即使文章继续支持国产替代，半导体第14且深度水下时仍必须回避。

7月20日数据验证了“板块趋势可以先于宽基修复”：A股宽基仍全部水下，但消费连续第1并满足相对强弱、连续性、量能和文章支持，因而可以独立升级主攻。与此同时，稳市力量买入双创并没有让科创50或半导体恢复资格；政策托底是环境解释，不是鱼盆硬条件。

7月21日数据反向验证了“单日大涨不能替代趋势位置”：科创50和半导体分别上涨10.73%和9.83%，且文章重新支持国产算力，但二者仍在后排和20日线下，所以继续回避；消费、创新药、煤炭虽保持前三，却因弱于A500统一降为趋势保持。文章补充的BOLL上下轨和MACD背离只用于观察短线进出时点，不改变这组中线结论。

7月22日数据出现了更接近中线修复的信号：上证50首次升至第3、站上20日线且强于A500，但连续性、量能和文章确认仍缺失，因此只能开启试探。煤炭升至第1并放量，却因文章中性无法主攻；这说明三条件只是方向资格，完整主攻仍需要连续性、量能和独立文章逻辑。

7月23日数据又撤销了上证50的一期试探，并让创新药跌出前三。更重要的方法变化来自文章时间尺度：作者长期看好国产算力，同时明确警惕长鑫IPO前的短期资金退出。本项目按中线趋势买入周期记录文章态度，因此科创50和半导体记为短期反对；长期产业判断不能被平均成中性，也不能覆盖后排水下的鱼盆结果。

### 2. AI主线

AI在文章中有两层含义：

- 股票主线：半导体、存储、光通信、算力、数据中心。
- 个人生产力：AI工具、WorkBuddy、代码生成、回测、知识库、工作原型。

这是理解作者的重要点：他不是只把AI当题材炒，而是把AI当成时代工具使用。所以他对AI长期共识偏乐观，但对短期硬件估值和二阶预期会转谨慎。

### 3. 流动性

美伊、油价、通胀、加息预期是6月中旬行情修复的关键链条：

```text
美伊缓和
→ 油价压力下降
→ 通胀预期下降
→ 加息预期下降
→ 流动性预期改善
→ 风险资产修复
```

这条链条解释了为什么作者在6月中旬从谨慎转向更积极。

### 4. 板块轮动

板块轮动的关键变化：

- 5月底：科技回调，消费/地产试探反弹。
- 6月中：科技重新占优。
- 6月底：半导体仍强，创新药开始出现。
- 7月初：创新药、证券、机器人进入前排，AI硬件出现二阶预期裂缝；7月9日半导体升到第1，7月10日创新药接替主攻；7月15日创新药和煤炭短暂主攻；7月17日主攻归零；7月20日消费独立升级主攻；7月21日主攻再次归零；7月22日上证50短暂试探；7月23日煤炭连续第1、红利新进第2，消费降至第3、创新药跌出前三，科技继续后排水下。

后续最应该盯的是确认项能否补齐：煤炭获得文章支持且不过热，红利补齐连续性和量能，消费恢复相对强势，创新药和上证50重新进入前三；科技需先消化IPO压力并站回20日线。

### 5. 交易纪律

作者最反复强调的纪律：

- 短线、中线、长线不能混。
- 博反弹失败不能改成长期价值投资。
- 单一个股信仰危险。
- 高位不敢买会错过主线，但高位追买也要有退出规则。
- 仓位管理比单次判断更重要。
- 不用贷款或其他高杠杆资金炒股，亏损后试图“捞回来”会让决策进一步失真。
- BOLL上下轨与MACD能量柱/背离可辅助观察短线时点，但不能把水下后排方向升级为趋势买入。
- 杠杆品种之间会通过保证金相互传染，可能迫使投资者卖出原本判断正确的仓位；方向判断不能替代账户流动性管理。
- 不熟悉的期货或跨资产品种少做或不做；量化时代的“抱团”叙事仍必须经过鱼盆硬条件过滤。
