# Web Instructions

这是项目结构化数据的只读展示层。运行本地服务时由助手启动并完成浏览器验收，不要求用户代为启动。

- 业务事实只能来自 `../docs/` 和 `../data/`，页面不定义阈值、不写回数据。
- `npm run dev` 和 `npm run build` 会先运行 `../scripts/build_web_snapshot.py`；项目校验失败时停止。
- 生成的 `public/data/project-snapshot.json` 和 `public/yupen-images/` 不进入版本控制。
- 页面必须明确展示文章日、指数数据日和板块数据日，不能合并成一个“更新时间”。
- 方向详情必须保留排名对比、三条件证据、文章态度、下一验证点、鱼盆原图和生命周期。
- 不新增实时行情、自动交易、泛资讯或单只股票建议。

视觉稿只约束布局、密度、层级和视觉语言；可见业务内容必须以真实快照为准。
